# Excercises

1. Write a recursive query which returns a number and its factorial for all
   numbers up to 10.

   E.g.,

````
   ------------
   | 0  | 1   |
   | 1  | 1   |
   | 2  | 2   |
   | 3  | 6   |
   | 4  | 24  |
   | 5  | 120 |
   ...

````

2. Write a recursive query which returns a number and the number in Fibonacci
   sequence at that position for the first  

   E.g.,

````
   ------------
   | 1  | 1   |
   | 2  | 2   |
   | 3  | 3   |
   | 4  | 5   |
   | 5  | 8   |
   | 6  | 13  |
   | 7  | 21  |
   ...

````
